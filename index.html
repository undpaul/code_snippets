<?code_snippets by undpaul

a code snippet repository for common usage in drupal installations
  • .gitignore for drupal projects 
    curl https://raw.github.com/undpaul/code_snippets/master/git/default.gitignore > .gitignore
  • .gitignore for drupal projects on freistilbox 
    curl https://raw.github.com/undpaul/code_snippets/master/git/freistilbox.gitignore > .gitignore
  • a bash file for executing a sites/all/scripts/update.sh from any place in the drupal folder 
    cd ~
curl https://raw.github.com/undpaul/code_snippets/master/sh/undpaul-update.sh > .drush/undpaul-update.sh
    Then just add a global alias in your .zshrc or .bash_profile: 
    alias uu="sh ~/.drush/undpaul-update.sh"
