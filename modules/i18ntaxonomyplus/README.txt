Dependencies
============
Internationalisation (Taxonomy translation): http://drupal.org/project/i18n

Module Description
==================
Adds
* Views Argument for filter against terms and their translations on the same time.
  (http://drupal.org/project/views)
* CCK Formater for content_taxonomy (http://drupal.org/project/content_taxonomy)
  to display the terms in the current selected language.
